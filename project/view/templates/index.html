About project

Info_21 version 2.0 is a progressive update of the original version, developed in Python using Flask and Jinja2, two popular tools for creating web applications. The program provides users with an intuitive interface for interacting with the PostgreSQL DBMS.

The main functions of Info_21 V.2 include viewing, editing, importing and exporting tables, executing custom SQL queries, and accessing and executing functions and procedures stored in the database. Thanks to this, users can easily and quickly access the necessary information, as well as manage data in the database.

Overall, Info_21 V.2 is a powerful and versatile PostgreSQL tool that provides flexibility and efficiency for developers, database administrators, and general users.
